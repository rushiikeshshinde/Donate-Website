# Donate-Website
This project is written in HTML, CSS, JavaScript (+JQuery) and PHP. 
It consists of : a home page, two donation pages (people and nature), a contact page and a login page.
In more detail, Donation pages: -contain payment forms,
                                -donations are stored in MySQL database.
                Contact page: uses SMTP protocol,
                              is implemented via PHPMailer library.
                Login page consists of sign up page and log out page.
